About SkyCast

SkyCast is a weather dashboard built to provide real-time weather updates, forecasts, and an interactive weather map. This project is part of my Unit 2 assessment for the Level 5 Diploma in Web Application Development.

The application combines modern web technologies with accessibility-first design to ensure all users can benefit from accurate and easy-to-understand weather data.
